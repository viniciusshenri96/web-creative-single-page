## 📖 About the Project

This project is a solution to the **Creative Agency Single-Page Site** challenge from Frontend Mentor.

The objective was to recreate the provided design as accurately as possible while ensuring responsiveness across different screen sizes.

The website includes smooth navigation, responsive layouts, reusable components, and clean semantic HTML.

---

- 📱 Fully Responsive Design
